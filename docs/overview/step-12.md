# Step 12. 여러 줄 코드 conflict 통합하기

`index.mjs`의 정렬 코드와 완료 건수 코드를 한 실행 흐름으로 합칩니다.

## 실습 순서

1. `branch/sort-tasks`를 만듭니다.
2. `main`에는 완료 건수 출력을 추가합니다.
3. 다른 branch에는 미완료 우선 정렬을 추가합니다.
4. merge해 여러 줄 conflict를 만듭니다.
5. import, 정렬, 집계, 반복문 순서로 최종 코드를 입력합니다.
6. 실행 뒤 해결 commit을 push합니다.

## 완료 기준

정렬과 완료 건수가 모두 동작하고 conflict marker가 사라졌으면 완료입니다.
