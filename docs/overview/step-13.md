# Step 13. 기능 branch 두 번 반복하기

대기 건수와 높은 우선순위 목록을 각각 별도 branch에서 구현합니다.

## 실습 순서

1. 새 기능 branch를 만듭니다.
2. `practice/task-app/index.mjs`에 기능 하나를 추가합니다.
3. Node.js로 실행하고 변경 내용을 commit합니다.
4. `main`으로 돌아와 `--no-ff`로 merge합니다.
5. `main`을 push하고 사용한 branch를 삭제합니다.
6. 두 번째 기능도 같은 순서로 반복합니다.

## 완료 기준

두 기능이 `main`에서 함께 실행되고 원격 저장소에 push되어 있으면 완료입니다.
