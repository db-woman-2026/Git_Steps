# Step 2. 변경 내용 비교하기

## 목표

파일을 수정했을 때 Git이 어떤 부분이 바뀌었는지 보여준다는 것을 확인합니다.

## 실습 순서

1. `practice/profile.md` 파일을 엽니다.
2. 관심사 또는 오늘의 연습 문장을 바꿉니다.
3. Source Control에서 파일을 클릭합니다.
4. VSCode diff 화면에서 지워진 줄과 추가된 줄을 확인합니다.
5. 변경 내용을 commit 합니다.

## VSCode에서 볼 것

- 빨간 줄은 사라진 내용입니다.
- 초록 줄은 새로 추가된 내용입니다.
- 아직 commit하지 않은 변경은 Source Control에 남아 있습니다.

## 완료 기준

수정한 문장이 diff에서 확인되고, 두 번째 commit으로 저장되어 있으면 완료입니다.
