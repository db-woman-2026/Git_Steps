# Step 14. 실행 결과와 GitHub 기록 확인하기

할 일 프로그램을 실행하고 완료 기록을 commit한 뒤 로컬과 원격 hash를 비교합니다.

## 실습 순서

1. 현재 branch와 작업 파일을 확인합니다.
2. 할 일 프로그램의 최종 출력을 확인합니다.
3. `practice/completion.md`를 직접 입력하고 commit합니다.
4. `main`을 push합니다.
5. `main`과 `origin/main`의 hash를 비교합니다.
6. GitHub 웹사이트에서 코드 파일과 commit 기록을 확인합니다.

## 완료 기준

GitHub에서 `practice/task-app`, `practice/completion.md`, 마지막 commit이 보이고 두 hash가 같으면 완료입니다.
