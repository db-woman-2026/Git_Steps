# Step 14. GitHub에 올리기

## 목표

로컬에 있는 Git 저장소를 GitHub에 올려 온라인에서도 확인합니다.

## 실습 흐름

1. GitHub에 빈 저장소를 만듭니다.
2. 로컬 저장소에 원격 주소를 연결합니다. 이미 `origin`이 있다면 내 개인 저장소 주소가 맞는지 먼저 확인합니다.
3. `main` 브랜치를 push 합니다.
4. GitHub 웹사이트에서 파일과 commit 기록을 확인합니다.

## VSCode에서 볼 것

- 원격 저장소는 온라인 백업 장소로 이해합니다.
- push 후 GitHub에서 같은 파일이 보이는지 확인합니다.
- 수강생 실습에서는 `main`만 push해도 충분합니다. `step-1` 같은 브랜치는 강사용 참고 브랜치입니다.

## 완료 기준

GitHub에서 `README.md`, `docs/overview`, `practice` 폴더와 commit 기록이 보이면 완료입니다.
