# Step 3. 첫 브랜치 만들기

## 목표

브랜치를 "원본을 건드리지 않고 따로 연습하는 공간"으로 이해합니다.

## 실습 순서

1. VSCode 왼쪽 아래의 브랜치 이름을 확인합니다.
2. 새 브랜치 `branch/profile-edit`을 만듭니다.
3. `practice/profile.md`에 문장을 하나 추가합니다.
4. 변경 내용을 commit 합니다.
5. `main`으로 돌아가서 방금 추가한 문장이 보이지 않는지 확인합니다.

## VSCode에서 볼 것

- 현재 브랜치 이름이 바뀌는지 확인합니다.
- 브랜치를 바꾸면 파일 내용도 달라질 수 있음을 확인합니다.

## 완료 기준

`branch/profile-edit`에는 새 문장이 있고, `main`에는 아직 그 문장이 없으면 완료입니다.
